Consumr

Media Library and Social Network

Drew Blumberg
March 13th, 2014

Project Goals:

  • Provide a user the ability to create a large media library (with pictures and slideshows) pulling from a large database of content.
  • Add a friendship system to be able to see what others are consuming.
  • Allow users to comment or make recommendations to their friends about their consumed content.
